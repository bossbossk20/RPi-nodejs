# Smart-Fridge-RPi-NodeJS

### Using NodeJS call python-shell
```
PythonShell.run('YourPythonCode.py', function (err) {
  if (err) throw err;
  console.log('finished');
});
console.log("test");
```
### How to install python-shell
```
$ npm install python-shell
```
### How to run
```
 $ npm install
 $ node index.js

```
